# Data Science Tools and Ecosystem Lab

This repository contains the Jupyter notebook for the **"Data Science Tools and Ecosystem"** lab, completed as part of the **IBM Data Science Professional Certificate** on Coursera.

## Notebook

- `DataScienceEcosystem.ipynb`  
  - Lists popular data science languages, libraries, and tools  
  - Includes examples of basic arithmetic expressions in Python  
  - Demonstrates use of markdown and code cells in Jupyter

## How to use

1. Download or clone this repository.
2. Open the notebook in Jupyter Notebook or JupyterLab.
3. Run the cells in order to reproduce the results.
